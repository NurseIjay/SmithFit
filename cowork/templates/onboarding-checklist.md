Title: Client Onboarding — Checklist

Purpose: Standard Cowork task template to finalize a new client onboarding in SmithFit.

Task fields:
- Owner: (assign coach)
- Due: (3 business days)

Checklist:
- [ ] Verify intake form and contact details
- [ ] Confirm allergies / restrictions in client folder
- [ ] Run `onboard-client` skill and add output to client folder
- [ ] Send welcome email + 7-day starter plan
- [ ] Create Cowork follow-up reminder (7 days)
- [ ] Add tags: `onboard`, `priority-new-client`

Notes:
- If any `flags` include `ESCALATE`, mark as medical escalation and notify clinical lead.
