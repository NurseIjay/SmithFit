### Why These First
1. **Concepts 01-03 talking heads:** Must test IJ on camera. This is the single biggest variable.
2. **01 B-roll:** Tests cinematic format against talking head -- reveals format preference.
3. **02 static + 03 carousel:** Tests non-video formats for TOF -- reveals format viability.
4. **08 quiz (2 formats):** Tests lead gen funnel. If CPL is low, this becomes the primary TOF channel.
5. **09 + 10 (static/carousel):** BOF retargeting converts warm traffic from TOF. Must be running simultaneously.
