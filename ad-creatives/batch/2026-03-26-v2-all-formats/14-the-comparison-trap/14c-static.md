# Concept 14: The Comparison Trap -- Static Image
**Funnel Stage:** TOF (Cold)
**Format:** Static Image Ad
**Objective:** Name the comparison shame, relieve it immediately, offer the shift-based reset

## Photo/Image Direction
- **Text-only design.** The visual progression from muted comparison (#B0B0B0) to the quiet question (#4A4A4A) to the giant gold "Nothing." (#C5A55A) creates an emotional arc without any images. The scale change IS the relief.
- **No photos needed.** Typography tells the story.
