How to Embed in GoHighLevel

Method 1: Custom HTML Block (Recommended)

  • ✓ Open your funnel page in GHL editor
  • ✓ Add a "Custom JS/HTML" element where you want the section
  • ✓ Copy the full contents of the artifact HTML file
  • ✓ Paste into the Custom HTML block
  • ✓ Remove the <html>, <head>, and <body> wrapper tags — keep only the <style>, the content <div>, and the <script>
  • ✓ Preview and adjust padding/margins to match your page flow

Method 2: iFrame Embed

  • ✓ Host the HTML file on your domain or a CDN (e.g., upload to GHL file manager)
  • ✓ Use an iframe element: <iframe src="YOUR_URL" style="width:100%; border:none; min-height:800px;"></iframe>
  • ✓ Add iframe auto-resize script for dynamic height adjustment
Important: When embedding multiple artifacts on the same page, namespace your CSS class names if you see style conflicts. Each artifact uses scoped class names to minimize conflicts, but if two sections share the same variable names (e.g., --gold), only include the :root variables once at the top of the page.
