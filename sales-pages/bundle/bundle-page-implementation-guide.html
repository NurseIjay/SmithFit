⚙️ GHL Embed Instructions

How to add each artifact to your live GHL page.

General Steps for Any Artifact

  1. Log into GHL page editor (hub.courseiq.app)
  2. Navigate to the Bundle page
  3. Click "Add Element" or "+" button
  4. Select "Custom HTML/Code" or "Raw HTML"
  5. Paste the full HTML contents of the artifact file
  6. Set the element width to Full Width
  7. Remove any default padding/margin on the container (set to 0)
  8. Save and preview

Success Metrics to Track

  • Conversion Rate: % of visitors who purchase a bundle (goal: +35–40% lift)
  • Average Order Value: Avg price paid when bundle is purchased (goal: +$150–200)
  • Page Time: Minutes spent on page (goal: increase from 2.5 → 3.5 min)
  • CTR on CTAs: Click-through rate on CTA buttons (goal: +20–25%)
  • Scroll Depth: % of users scrolling past the fold (goal: 75%+ reach commitment trust section)
  • FAQ Interactions: % of users opening FAQ questions (goal: 40%+)
